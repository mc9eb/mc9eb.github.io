## Guess the Number Game

A simple interactive web game where players try to guess a randomly generated number

## Features

### Basic Features
- Random number generation
- User input system
- Feedback (too high / too low)

### Enhanced Features
- Score tracking
- Limited attempts
- Hint system (even/odd)

### Advanced Features
- Difficulty levels (Easy, Medium, Hard)
- Persistent leaderboard using local storage

## How to Play
1. Select a difficulty
2. Click "Start Game"
3. Enter your guess
4. Use hints if needed
5. Try to guess before running out of attempts!
